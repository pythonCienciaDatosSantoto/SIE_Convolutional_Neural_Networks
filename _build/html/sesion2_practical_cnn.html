Construyendo Nuestra Primera CNN#

Objetivo: Aplicar los conceptos teÃ³ricos de la primera parte de este cursillo para construir y entrenar una Red Neuronal Convolucional (CNN) simple. Compararemos directamente sus resultados con los de las redes densas de la SesiÃ³n 1 para el problema de clasificaciÃ³n de CIFAR-10.

Comenzamos importando las librerÃ­as necesarias. AdemÃ¡s de las que ya usamos, ahora importaremos las capas especÃ­ficas de las CNNs: Conv2D y MaxPooling2D.

Este paso es casi idÃ©ntico al de la SesiÃ³n 1. Cargamos CIFAR-10, definimos los nombres de las clases, normalizamos las imÃ¡genes y aplicamos codificaciÃ³n one-hot a las etiquetas.

La diferencia crucial: Â¡NO vamos a aplanar las imÃ¡genes! Las capas convolucionales necesitan la informaciÃ³n espacial, por lo que la entrada al modelo mantendrÃ¡ su forma original de (32, 32, 3).

3. ConstrucciÃ³n del Modelo CNN#

Ahora viene la parte emocionante. Construiremos una arquitectura de CNN simple pero efectiva. Seguiremos el patrÃ³n clÃ¡sico:

  1. Bloque Convolucional 1: Una capa Conv2D para detectar 32 tipos de caracterÃ­sticas, seguida de una capa MaxPooling2D para reducir el tamaÃ±o.

  2. Bloque Convolucional 2: Otra capa Conv2D para aprender caracterÃ­sticas mÃ¡s complejas (64 filtros), seguida de otro MaxPooling2D.

  3. Bloque de ClasificaciÃ³n: Aplanamos (Flatten) la salida del Ãºltimo bloque y la pasamos a una capa Dense para la clasificaciÃ³n final.

model_cnn = Sequential([
    # --- Bloque de ExtracciÃ³n de CaracterÃ­sticas ---
    
    # Capa Convolucional 1: 
    # 32 filtros (detectores de caracterÃ­sticas) de tamaÃ±o 3x3.
    # 'padding="same"' asegura que la salida tenga el mismo ancho y alto que la entrada.
    # 'activation="relu"' es la funciÃ³n de activaciÃ³n estÃ¡ndar.
    # 'input_shape' solo se especifica en la primera capa.
    Conv2D(32, (3, 3), padding='same', activation='relu', input_shape=(32, 32, 3)),
    MaxPooling2D((2, 2)), # Reduce a la mitad el tamaÃ±o del mapa de caracterÃ­sticas (de 32x32 a 16x16).
    
    # Capa Convolucional 2:
    # Aumentamos el nÃºmero de filtros a 64 para aprender patrones mÃ¡s complejos.
    Conv2D(64, (3, 3), padding='same', activation='relu'),
    MaxPooling2D((2, 2)), # Reduce de nuevo a la mitad (de 16x16 a 8x8).
    
    # --- Bloque de ClasificaciÃ³n ---
    
    # Aplanamos los mapas de caracterÃ­sticas 2D a un solo vector 1D.
    Flatten(),
    
    # Una capa densa para realizar la clasificaciÃ³n basada en las caracterÃ­sticas extraÃ­das.
    Dense(64, activation='relu'),
    
    # Capa de salida con 10 neuronas (una por clase) y activaciÃ³n softmax.
    Dense(num_classes, activation='softmax')
])
